Bienvenido a la guía del paradigma funcional con SML

Este software es desarrollado por la Universidad tecnológica de Pereira, el cual busca poder enseñar el paradigma de programación declarativo funcional mediante el lenguaje de programación StandarML. Se ha escogido este paradigma de programación funcional, debido a que es un paradigma enfocado a la enseñanza de la programación, al igual que muchos otros paradigmas, con el paradigma funcional se puede llegar a lograr construir programas robustos y complejos con el fin de poder solucionar alguna necesidad, entre sus muchas aplicaciones se destacan aplicaciones para técnicas y matemáticas, Inteligencia artificial, compiladores o analizadores, entre otras muchas.

¿Que es SML?

SML es un lenguaje de programación de tipo seguro que incorpora muchas ideas innovadoras en el diseño de lenguajes de programación. Es un lenguaje estáticamente tipado con un sistema de tipos extensible. Soporta la inferencia de tipos polimórficos que prácticamente elimina la carga de especificar los tipos de las variables y facilita enormemente la reutilización del código. Proporciona una gestión automática y eficaz del almacenamiento de las estructuras de datos y las funciones.

¿Qué es la programación funcional?

El nombre ya lo sugiere: la programación funcional o functional programming se centra en las funciones. En un programa funcional, todos los elementos pueden entenderse como funciones y el código puede ejecutarse mediante llamadas de función secuenciales. Por el contrario, no se asignan valores de forma independiente. Una función se imagina mejor como una variante especial de un subprograma. Esta es reutilizable y, a diferencia de un procedimiento, devuelve directamente un resultado.

PUNTOS A TOCAR EN ESTE SOFTWARE

Con el objetivo de poder cumplir con los criterios y formatos de evaluación, se menciona a continuación cuales objetivos son los que este software abarca:

  • ROBUSTEZ: Este software cuenta con un manual de usuario y un manual técnico el cual le permite tanto a la persona que lo va a usar como el que quiere saber como esta desarrollado el mismo tener una visión clara sobre el software y sus secciones más significativas
  • EXTENDIBILIDAD: A pesar de que este software no cuenta con una facilidad de añadírseles nuevas características, se cuenta con la documentación adecuada para poder implementar nuevas propiedades
  • MANTENIMIENTO: Se presenta una buena documentación para realizar cambios en el mismo
  • GRADO DE INVENCIÓN O INNOVACIÓN: A pesar de que SML es un lenguaje conocido, es difícil encontrar tutoriales y ejemplos de este lenguaje de programación, adicional de eso, este software cuenta con un sistema de ejemplos los cuales muestran el paso a paso de diferentes ejercicios aplicados en el lenguaje

¿Que son los módulos?

Los módulos son secciones o apartados los cuales se tocan, se explican y se realizan ejercicios y ejemplos de diferentes temas del lenguaje SML, cada módulo cuenta con ejemplos los cuales tienen un paso a paso implementado, es de esta manera es como se busca poder que el usuario entienda cada uno de los temas tocados.
