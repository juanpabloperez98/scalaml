Ejemplo 1

Escribir una función recursiva que calcule el factorial de un número n donde n >= 1 y da como resultado el producto de todos los enteros comprendidos entre 1 y n. 

                        val x = valor_ingresado;
    fun factorial(x) =
        if x = 1 then
            1
        else
            factorial(x-1)*x;
    factorial(x);
Paso a paso Siguiente Ver resultado 
                         Se define la variable x y es igualada al valor ingresado por el usuario
 Luego se declara la función factorial que recibe como parámetro una variable x                        
 Dentro de la función se valida la base (recordar que la base es el condicional que determina cuándo ha finalizado la recursión), donde se valida que la variable x sea igual a 1
 Si x es igual a 1 entonces la función retorna 1
 De lo contrario si x no es igual a 1
 Se volverá hacer el llamado a la función factorial pero pasándole como parámetro ahora la variable x - 1
 Se hace el llamado a la función factorial declarada anteriormente, pasándole como parámetro la variable x.
