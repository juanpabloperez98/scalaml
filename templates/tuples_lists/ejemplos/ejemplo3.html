Ejemplo 3

Dada tres tuplas con diferentes elementos dentro de cada una, realizar un programa que concatene solamente los elementos de tipo string 

                        val tupla1 = (2.0,11,"Programacion");
    val tupla2 = (#"A","en",20 div 2);
    val tupla3 = ("ML",2+1,20 div 2);
    val concat_ = #3(tupla1) :: #2(tupla2) :: #1(tupla3) :: nil;
    val concat_result = concat(concat_);
Paso a paso Siguiente Ver resultado

 Se crea una lista de los elementos de tipo string que hay en las tuplas 1 2 y 3 (Nótese que se accede a dichos elementos utilizando la notación #), el resultado es almacenado en una variable llamada concat_
 Ahora teniendo la lista en la variable concat_ se utiliza la función concat para formar un string con todos los elementos de la lista
