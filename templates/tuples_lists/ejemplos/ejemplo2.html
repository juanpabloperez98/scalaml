Ejemplo 2

Dada una palabra ingresada por el usuario (no más de 4 caracteres de longitud) se pide realizar un programa que pueda coger cada una de las letras de la palabra convierta a su correspondiente código ASCII y haga la suma de dicha palabra. 

                        val word = numero_ingresado;
    val convert_word = explode(word);
    val firts_element = ord(hd(convert_word));
    val second_first = ord(hd(tl(convert_word)));
    val third_first = ord(hd(tl(tl(convert_word))));
    val four_first = ord(hd(tl(tl(tl(convert_word)))));
    val suma = firts_element + second_first + third_first + four_first;
Paso a paso Siguiente Ver resultado 
                         Se define la variable llamada word la cual va almacenar el valor ingresado
 Luego se define la variable convert_word y se iguala al valor retornado por la función explode (NOTA: recordar que la función explode convierte un string a una lista de caracteres)                        
 Se obtiene el primer elemento de la lista y utilizando la función ord se obtiene el código ASCII correspondiente del elemento devuelto, y este se almacena en la variable firts_element                        
 Se obtiene el segundo elemento de la lista y utilizando la función ord se obtiene el código ASCII correspondiente del elemento devuelto, y este se almacena en la variable second_element
 Se obtiene el tercer elemento de la lista y utilizando la función ord se obtiene el código ASCII correspondiente del elemento devuelto, y este se almacena en la variable third_element
 Se obtiene el cuarto elemento de la lista y utilizando la función ord se obtiene el código ASCII correspondiente del elemento devuelto, y este se almacena en la variable four_element
 Por último se suman todas las variables definidas anteriormente (firts_element, second_element, third_element, four_element)
