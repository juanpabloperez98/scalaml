Ejemplo 2

Realizar un programa que dependiendo un número definido por el programador valide si el número es mayor o igual a 5 entonces realice una multiplicación entre el número definido y el número 5, pero si es menor a 5 entonces realice una resta entre el número 5 y el número definido. NOTA: el número que ingrese el usuario no puede ser menor o igual a cero. 

                        val num = num_ingresado;
    if num <= 0 then
        0
    else
        if num >= 5 then num*5
        else 5 - num
Paso a paso Siguiente Ver resultado 
                         Primero se define la variable num y se iguala a un valor entero definido por el programador. Nota: en su computador deberá ser un número definido por usted, esto debido a que aun no se han explicado los temas de entradas y salidas en SML
 Luego se valida con el condicional if, que el número no sea menor a cero                        
 De ser menor a cero entonces se retornará como resultado el número cero Nota: en cualquiera de los casos siempre se debe retornar el mismo tipo de valor es decir siempre de deberá retornar un Integer                        
 De no ser menor a cero                        
 Se valida ahora si el número es mayor o igual a 5 de ser así, entonces se multiplica el valor de la variable num por el número 5                        
 Si por el contrario el valor de la variable num es menor a 5 entonces se realizará la resta entre 5 y el valor de la variable num
