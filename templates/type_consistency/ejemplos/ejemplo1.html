Ejemplo 1

Escriba un programa el cual declare 2 variables de tipo reales (los valores que usted considere) y haga la división entera de dicha variables considerando

  • Que la variable 1 debe ser convertida al entero superior
  • La variable 2 se le deben eliminar los decimales que tengan
NOTA: como aún no se ha visto la sección de reglas de entradas y salidas, usted en su código local declare las variables del ejemplo y dele valores estáticos 

                        val v1 = valor_ingresado;
    val v2 = valor_ingresado;
    val resultado = ceil(v1) + trunc(v2);

 Se define la primera variable con el nombre v2                        
 Se utiliza la función trunc pasándole como argumento la variable v1 y se le suma el valor de aplicar la función ceil pasandole como parametro la variable v2
