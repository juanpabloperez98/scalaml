Ejemplo 2

Desarrolle un programa tal que permita definir la media aritmética de 3 números. NOTA: como aún no se ha visto la sección de reglas de entradas y salidas, usted en su código local declare las variables del ejemplo y dele valores estáticos 

                        val x:int = valor_ingresado;
    val y:int = valor_ingresado;
    val z:int = valor_ingresado;
    val media:int = x+y+z div 3;
Paso a paso Siguiente Ver resultado 
                         Se declara el primer enlace de tipo entero o int, el cual se le coloca como nombre x y se iguala al valor ingresado por el usuario.
 Se declara el segundo enlace de tipo entero o int, el cual se le coloca como nombre Y y se iguala al valor ingresado por el usuario.                        
 Se declara el segundo enlace de tipo entero o int, el cual se le coloca como nombre Z y se iguala al valor ingresado por el usuario.                        
 Se declara el ultimo enlace medio el cual es el resultado de la operación media. Nota: La operación media es la suma de los valores ingresados divido en su total
