Defina una función que se llama cube el cual calculará el cubo de un número real ingresado por el usuario 

                        val num = valor_ingresado;
    fun cube(x:int) = x*x*x;
    cube(num);
Paso a paso Siguiente Ver resultado 
                         Primero se declara la variable num y se iguala al valor ingresado
 Luego se define la función cube, esta función recibe como parámetro una variable x (se le define que es de tipo entero) y en el cuerpo de la función se define la operación que va realizar dicha función (La multiplicación 3 veces del parámetro)                        
 Por último se hace el llamado a la función cube pasándole como parámetro la variable num
