Ejemplo 3

Construir una función que reciba una lista y retorna el reverso de la lista (longitud de 3) 

                        val lista = lista_ingresada;
    fun reverse xs =
        hd(tl(tl(xs))) :: hd(tl(xs)) :: hd(xs) :: nil;
    reverse(lista);
Paso a paso Siguiente Ver resultado 
                         Se crea una variable llamada lista y se igual al valor ingresado por el usuario
  Luego se define la función reverse la cual recibe como parámetro una variable xs (Esta variable va a representar la lista que se va a pasar por parámetro)                        
  Luego dentro del cuerpo de la función se obtiene el último elemento de la lista, y se concatena con el elemento medio de la lista y por último se concatena con el primer elemento de la lista (En otras palabras se altera el orden de la lista)
  Al final se llama a la función reverse pasándole como parámetro la variable lista)
