Ejemplo 2

Realice una función que recibe una tupla como parámetro y devuelva el número menor de los elementos de dicha tupla 

                        val tupla = tupla_ingresada;
    fun lesser_to_numbers(x,y) = 
        if x < y then x else y;
    fun lesser(x,y,z) = 
        lesser_to_numbers(lesser_to_numbers(x,y),z);
    lesser(tupla);
Paso a paso Siguiente Ver resultado 
                         Se declara la variable tupla y se iguala al valor que ingresa el usuario
 Luego se declara una función llamada lesser_to_numbers la cual recibe dos parámetros (x,y). Esta función lo que devuelve es el menor entre dos números                        
 Dentro del cuerpo de la función se valida si x es menor que y entonces devuelve x, si no es mayor entonces devolverá y                        
 Luego se define la función lesser, esta función va a recibir 3 parámetros (x,y,z)
 Dentro del cuerpo de dicha función vamos hacer el llamado a la función lesser_to_numbers pasándole como argumentos, un llamado a la misma función lesser_to_numbers con los parámetros (x,y) y la variable z (Esto es un llamado recursivo, el cual lesser_to_numbers con los parámetros (x,y) devuelve el valor menor de x o y, y después retorna el valor menor entre el anterior resultado y z)
 Por último se hace el llamado a la función lesser y se le pasa como parámetro la variable tupla (Recordar que esta variable va a tener almacenado las 3 variables que se reciben como argumento)

Ingrese tupla

Nota para ingresar una tupla es necesario escribir los parentesís
